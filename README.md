# OP工作室 博客网站

AI自动化与开发者生产力技术博客 - 1:1 还原 [jangwook.net](https://jangwook.net/zh/)

## 技术栈

- **框架**: Next.js 15 (App Router)
- **数据库**: Neon PostgreSQL
- **样式**: Tailwind CSS
- **语言**: TypeScript
- **认证**: Neon Auth + Stack Auth
- **部署**: Vercel

## 开始开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 启动生产服务器
npm start

# 代码格式化
npm run format

# 代码检查
npm run lint
```

## 项目结构

```
mysite/
├── app/                 # Next.js App Router
├── src/                 # 源代码
│   ├── components/      # React 组件
│   ├── lib/            # 工具函数
│   ├── hooks/          # 自定义 Hooks
│   ├── types/          # TypeScript 类型
│   └── i18n/           # 国际化配置
├── public/             # 静态资源
└── docs/               # 项目文档
```
